Monarch Bot

Monarch Bot is a free and open source software to make hosting tournaments on Cockatrice easier. This is the minimum required version. In the client you can go to Help>General then, set Update channel to Beta.

Most Common Issue

The most common issue is having an out of date Cockatrice client which means that you cannot see the games that are made by the bot. If this is not your issue to to the faq page or create an issue on github.
